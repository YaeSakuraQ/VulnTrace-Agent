# Kali 工具基线

- `nmap -sn` 适合低风险主机发现。
- `nmap -sV --version-light` 适合受控服务识别。
- `curl` 或安全的 HTTP 请求库适合页面标题、响应头、状态码抓取。
- `ffuf` 适合在极小词表、低并发和限时条件下做补充型内容枚举。
- `nikto` 适合作为教学靶场中的受控 Web 验证主工具，但应走审批，并按目标画像收敛参数。
- 对特定靶场可在 `nikto` 之后补一条极小的专用验证，例如 `mini_httpd` 的空 `Host` 原始请求，用于判断漏洞前提是否真的满足。

## 使用约束

- 不允许让模型直接拼接 Shell。
- 必须使用固定模板、参数校验、超时控制和输出截断。
- 工具输出应落盘保存，并在数据库中记录摘要路径。
